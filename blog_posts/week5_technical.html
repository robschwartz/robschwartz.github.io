Getting into Ruby classes has been a small war with many little battle wins. The first battle was understanding why we use Ruby classes.

Ruby classes are a fantastic way to create new objects with a pre-loaded set of methods (Learning about classes in CSS helped me understand this as well since I'm more of a visual learner). Instead of defining a method for each new object you implement in the code, you can just create the object into the Class and all the methods will be pre-loaded. I know, not very clear, but hopefully an everyday example will help explain it a bit better.
